contentEditable

用途

contentEditable 属性用来表明一个元素是否处于可编辑的状态

语法

value 的值为：

  • true 表示该元素可编辑.
  • false 表示元素不可编辑.
  • inherit 表示该元素继承了其父元素的可编辑状态.
