使用SVG来制作变形补间动画

开始之前，先来看看下面的示例：

从上面到示例中可以看到，按钮两个状态之间的转换是有一个补间的过度效果，Google在他们新推出的设计规范material design中，大量使用了这种补间的动画效果。这种补间动画效果确实能使产品的体验更加的细腻。正所谓随风潜入夜，润物细无声。随着CSS3技术的发展以及浏览器变得越来越强劲，在web网页开发中补间动画应用也应用的越来越多。

当然一些简单的补间动画使用css3种的transform也能实现，不过transform变换就那几种，局限也是比较大的。最好的实现其实使用svg来实现，下面我们就来看看怎样使用svg实现一个按钮的补间动画。

动画是使用svg自带的SMIL动画技术来实现的，至于SMIL动画的一些基本介绍是啥，可以去上一篇文章有详细的介绍。

形状之间必须有相同的控制点

我们都知道，svg形状都是又不同控制点之间的路径构成的。而要实现图标之间的补间动画，图标需有相同的控制点，不然补间动画就无法实现。

所以我们在设计图标的时候，最好是在一个单一的路径里面编辑。

SMIL来实现补间动画效果

1、设计图标

使用Adobe Illustrator等矢量设计软件设计我们需要的矢量图形。我们这个例子是要从一个星星图标和勾之间来做一个补间动画效果。下图是星星的形状：

设计完之后，拷贝这个星星的svg代码到一个新的文件中来设计下一个图标。这样就可以保证它们之间控制点的个数是相同的。

2、然后以上一个图标的坐标点为基础来变化为新的图标

其实在矢量设计软件中，就是拖拽这些控制点来变化为各种形状，如下图所示的过程：

最终变成了一个勾的形状：

3、星星形状的svg代码

<svg viewBox="0 0 194.6 185.1">

      <polygon fill="#FFD41D" points=" ... shape 1 points ... ">

      </polygon>

</svg>

4、添加要运动的animate元素，指定要运动属性

<svg viewBox="0 0 194.6 185.1">

      <polygon fill="#FFD41D" points=" ... shape 1 points ... ">

    <animate attributeName="points" dur="500ms" to=" ... shape 2 points ... " />

  </polygon>

</svg>

这里制定的动画效果是在网页加载的时候就会运行的，所以我们要使用一点点js来控制动画的状态。

5、给动画效果绑定点击事件来触发动画效果

SMIL也支持诸如点击和鼠标滑过等事件，所以我们可以指定在点击的时候才触发动画效果：

<polygon id="shape" points=" ... shape 1 points ... ">

   <animate begin="shape.click" attributeName="points" dur="500ms" to=" ... shape 2     points ... />

</polygon>

上面的代码也有它的局限性，你只能在点击svg的时候，才触发动画效果。不过这里的svg是按钮的一部分，我们是希望在点击按钮的时候就触发svg的动画效果。

我们可以直接给animate元素一个ID，使用javascript很容易对它进行控制：

<animate id="animation-to-check" begin="indefinite" ... />

结合SMIL提供的beginElement()方法(beginElement()方法是执行animate元素中所定义的动画方法)。

animationToCheck = document.getElementById("animation-to-check");

// 点击就会触发ID为animation-to-check中定义的动画效果
animationToCheck.beginElement();
